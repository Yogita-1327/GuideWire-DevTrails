# GigGuard

## AI-Powered Income Protection for Gig Delivery Workers

## Problem Statement
Gig delivery workers rely on daily work for income, making them highly vulnerable to sudden disruptions. Events like heavy rain, heat waves, pollution, curfews, and app downtime can reduce their working hours and cause immediate income loss.

Key challenges include:

-No insurance focused on income disruption

-Unpredictable earning loss during external events

-Lack of affordable weekly protection

-Delayed or complicated claim processes

GigGuard aims to solve this gap with an AI-powered, automated income protection system.

## Solution
GigGuard provides a smart income protection system that:

- Assesses disruption risk using AI
- Offers low-cost weekly protection plans
- Detects disruption events automatically
- Triggers claims without manual effort
- Supports quick and transparent payouts

## Key Features

### AI Risk Assessment
Predicts the likelihood of disruptions based on weather, pollution, and location data.

### Weekly Pricing Model
Provides affordable weekly coverage plans suitable for gig workers.

### Parametric Insurance Triggers
Uses predefined conditions such as rainfall, AQI level, heat threshold, or app downtime to trigger claims automatically.

### Automated Claims
Eliminates long claim forms by detecting events in real time.

### Instant Payout Support
Enables fast payout processing once disruption conditions are verified.

### Fraud Detection
Uses activity and location validation to reduce false claims.

## How It Works
1. Worker registers on GigGuard.
2. AI calculates the worker’s disruption risk score.
3. A suitable weekly protection plan is generated.
4. External APIs monitor disruption conditions.
5. If a disruption threshold is crossed, the claim is triggered automatically.
6. The worker receives payout support for income loss.

## Target Users
- Food delivery workers
- Grocery delivery partners
- Hyperlocal logistics workers
- Other gig workers dependent on daily task-based earnings

## Example Disruption Triggers
- Heavy rainfall above a defined threshold
- Extreme heat above a safe working limit
- AQI crossing a high pollution level
- Flood or curfew alerts in the worker’s area
- Platform outage affecting delivery operations

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript / React
- **Backend:** Node.js / Flask
- **Database:** MongoDB / Firebase
- **AI/ML:** Python, Scikit-learn
- **APIs:** Weather API, AQI API, Maps API
- **Payments:** Razorpay / UPI integration

## Innovation
GigGuard goes beyond traditional insurance by focusing specifically on **income disruption** instead of health or asset coverage. It combines:

- AI-based risk prediction
- Weekly micro-insurance pricing
- Parametric event triggers
- Automated claim settlement

This makes it faster, more accessible, and better suited for gig workers.
